# 🎲 LanzarDado (Dice Roller) – Android App Simulation

🌍 *[Read in English](#-english) | [Leer en Español](#-español)*

---

## 🇺🇸 English

A native Android application built with Kotlin that simulates rolling a dice. 

This project was created to deliver a quick and reliable random number generation tool, ideal for board games, making decisions, or studying core Android development concepts.

---

### 📖 About the Project
**LanzarDado** is a mobile application focused on simplicity and performance. 
The goal of the project was to design a straightforward **user experience** where users can interact with a virtual dice and get immediate visual feedback.

The system works entirely locally on the device, using efficient randomization algorithms to ensure fair results on every roll.

---

### 🚀 Main Features
🎲 **Instant Generation:** Roll a virtual dice and get a number from 1 to 6 instantly.
⚡ **Performance-Oriented:** Lightweight application with zero lag between interactions.
📱 **Responsive UI:** Adapts to different screen sizes and orientations on Android devices.
🎨 **Visual Feedback:** Clear and dynamic UI updates corresponding to the dice result.

---

### 🧠 Application Logic
The core logic of the system is implemented using **Kotlin**. 

When the user taps the roll button, the application:
1. Triggers an event listener.
2. Calls a randomization function `(1..6).random()`.
3. Updates the UI thread dynamically to display the corresponding image/value.
4. Manages the state of the view to ensure a smooth transition.

---

### 🎨 UI & User Experience
The project focuses on **modern mobile UX principles**.

* **Intuitive Design:** No learning curve; the primary action (rolling the dice) is immediately clear.
* **Native Components:** Built using standard Android UI components for a native look and feel.
* **Accessibility:** High contrast and large interaction areas for comfortable use.

---

* **Platform:** Android
* **Language:** Kotlin
* **IDE:** Android Studio
* **Version Control:** Git & GitHub

## 🇪🇸 Español

Una aplicación nativa de Android construida con Kotlin que simula el lanzamiento de un dado.

Este proyecto fue creado para ofrecer una herramienta rápida y confiable de generación de números aleatorios, ideal para juegos de mesa, toma de decisiones o para estudiar conceptos básicos de desarrollo en Android.

---

### 📖 Acerca del Proyecto
**LanzarDado** es una aplicación móvil enfocada en la simplicidad y el rendimiento.
El objetivo del proyecto fue diseñar una **experiencia de usuario** directa, donde los usuarios puedan interactuar con un dado virtual y obtener retroalimentación visual inmediata.

El sistema funciona de manera completamente local en el dispositivo, utilizando algoritmos eficientes de aleatoriedad para garantizar resultados justos en cada lanzamiento.

---

### 🚀 Características Principales
🎲 **Generación Instantánea:** Lanza un dado virtual y obtén un número del 1 al 6 al instante.
⚡ **Alto Rendimiento:** Aplicación ligera sin demoras entre interacciones.
📱 **Interfaz Responsiva:** Se adapta a diferentes tamaños de pantalla y orientaciones en dispositivos Android.
🎨 **Retroalimentación Visual:** Actualizaciones de interfaz claras y dinámicas que corresponden al resultado del dado.

---

### 🧠 Lógica de la Aplicación
La lógica central del sistema está implementada usando **Kotlin**.

Cuando el usuario presiona el botón de lanzar, la aplicación:
1. Activa un *event listener* (escuchador de eventos).
2. Llama a una función de aleatoriedad `(1..6).random()`.
3. Actualiza el hilo de la interfaz de usuario (UI) dinámicamente para mostrar la imagen/valor correspondiente.
4. Gestiona el estado de la vista para asegurar una transición fluida.

---

### 🎨 UI y Experiencia de Usuario
El proyecto se enfoca en **principios modernos de UX móvil**.

* **Diseño Intuitivo:** Sin curva de aprendizaje; la acción principal es evidente de inmediato.
* **Componentes Nativos:** Construido usando componentes de UI estándar de Android para una sensación nativa.
* **Accesibilidad:** Alto contraste y áreas de interacción amplias para un uso cómodo.
